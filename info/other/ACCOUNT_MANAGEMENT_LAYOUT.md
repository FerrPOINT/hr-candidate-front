## 📱 API ИНТЕГРАЦИЯ

### 🔗 ENDPOINTS
- `GET /users` - Список пользователей с пагинацией
- `POST /users` - Создание пользователя
- `GET /users/{id}` - Получение пользователя
- `PUT /users/{id}` - Обновление пользователя
- `DELETE /users/{id}` - Удаление пользователя
